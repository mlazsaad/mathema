القيمة المطلقة (Absolute Value)

بسم الله الرحمن الرحيم

🎯 أهداف الدرس

بنهاية هذا الدرس سيكون الطالب قادرًا على:

  • فهم معنى القيمة المطلقة
  • تبسيط التعابير التي تحتوي على قيمة مطلقة
  • حل معادلات من النوع |x| = a و |x| = |b|
  • التمييز بين الحالات المختلفة
  • تمثيل المعادلات على خط الأعداد
  • حل معادلات أكثر تعقيدًا تتضمن خطوات إضافية
  • فهم كيف تعالج القيمة المطلقة الإشارات السالبة والموجبة

🧠 أولاً: ما هي القيمة المطلقة؟

القيمة المطلقة لأي عدد هي المسافة بين هذا العدد والصفر على خط الأعداد.

🔹 لا تُعبر عن اتجاه، بل عن مقدار فقط.

🟧 تعريف رياضي (Mathematical Definition):

|x| = { x إذا كان x ≥ 0 -x إذا كان x < 0

📌 أمثلة (Examples):

|5| = 5 |-3| = 3 |0| = 0

🔢 ثانياً: حل المعادلات البسيطة للقيمة المطلقة (Solving Simple Absolute Value Equations)

✳️ الشكل الأساسي (Basic Form): |x| = a

إذا كانت a > 0، فالحل هو حالتان:

x = a أو x = -a

🧪 مثال 1 (Example 1):

|x| = 7 ⇒ x = 7 أو x = -7

❗ ملاحظة مهمة (Important Note):

إذا كانت القيمة سالبة مثل |x| = -5، فلا يوجد حل.

لأن القيمة المطلقة لا يمكن أن تساوي عددًا سالبًا.

🧩 ثالثاً: حل معادلات مثل |x-c| = a (Solving Equations Like |x-c| = a)

🧪 مثال 2 (Example 2):

|x-4| = 6 ⇒ x-4 = 6 أو x-4 = -6 ⇒ x = 10 أو x = -2

🔧 رابعاً: خطوات عامة لحل معادلات القيمة المطلقة (General Steps for Solving Absolute Value Equations)

  • عزل القيمة المطلقة في جهة واحدة (Isolate Absolute Value)
  • إنشاء حالتين بدون رمز القيمة المطلقة (Create Two Cases)
  • حل كل معادلة على حدة (Solve Each Case)
  • التحقق من الحل عند الحاجة (Verify Solution)

🧮 خامساً: أمثلة متنوعة (Various Examples)

🧪 مثال 3 (Example 3):

|2x+1| = 5 ⇒ 2x+1 = 5 أو 2x+1 = -5 ⇒ x = 2 أو x = -3

🧪 مثال 4 (Example 4):

|3x-4| = 10 ⇒ 3x-4 = 10 أو 3x-4 = -10 ⇒ x = 14/3 أو x = -2

❌ مثال 5 (Example 5):

|x + 2| = -7

لا يوجد حل، لأن القيمة المطلقة لا تساوي عددًا سالبًا.

🔁 سادساً: معادلات من النوع |A| = |B| (Equations of Type |A| = |B|)

🧪 مثال 6 (Example 6):

|x+1| = |3x-5| ⇒ x+1 = 3x-5 أو x+1 = -(3x-5) ⇒ x = 3 أو x = 1

📊 سابعاً: تمثيل المعادلات على خط الأعداد (Representing Equations on Number Line)

المعادلات |x-a| = b تمثل نقطتين تبعدان مسافة b عن a.

مثال: |x-2| = 5

⇒ الحل: x = 7 أو x = -3

نرسم نقطتين على خط الأعداد عند 7 و -3.

📚 ثامناً: تطبيقات حياتية على القيمة المطلقة (Real-Life Applications of Absolute Value)

مثال:

إذا كانت درجة الحرارة الآن 8°، وتغيرت بمقدار معين فأصبحت تبعد عن الصفر بـ 8 درجات، ما هي القراءة الجديدة؟

|x| = 8 ⇒ x = 8 أو x = -8

🧠 ملخص الدرس (Lesson Summary)

  • القيمة المطلقة تعني "المسافة عن الصفر"
  • لا يمكن أن تساوي قيمة سالبة
  • نحول المعادلة إلى حالتين عند إزالة القيمة المطلقة
  • نتحقق دائمًا من وجود حل
  • معادلات القيمة المطلقة تظهر كثيرًا في مسائل الحياة

🏠 واجب منزلي (Homework)

  3. |x+4| = |x-2|
