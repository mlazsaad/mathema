المتطابقات التربيعية والتكعيبية (Quadratic & Cubic Identities)

بسم الله الرحمن الرحيم

المتطابقات التربيعية (Quadratic Identities)

المتطابقة الأولى: مربع مجموع حدين

(a + b)² = a² + 2ab + b²

المتطابقة الثانية: مربع فرق حدين

(a - b)² = a² - 2ab + b²

المتطابقة الثالثة: جداء مجموع حدين في فرقهما

(a + b)(a - b) = a² - b²

المتطابقات التكعيبية (Cubic Identities)

ملاحظة هامة (Important Note)

المتطابقات التكعيبية لم تستخدم في المنهاج السوري ولكن من المهم التعرف عليها.

المتطابقة الأولى: مكعب مجموع حدين (Cube of Sum)

(a + b)³ = a³ + 3a²b + 3ab² + b³

ملاحظات:

  • الحد الأول: مكعب الأول (a³)
  • الحد الثاني: ثلاثة أضعاف مربع الأول في الثاني (3a²b)
  • الحد الثالث: ثلاثة أضعاف الأول في مربع الثاني (3ab²)
  • الحد الرابع: مكعب الثاني (b³)

مثال: (x + 2)³

  1. a = x, b = 2
  2. x³ + 3x²(2) + 3x(2²) + 2³
  3. x³ + 6x² + 12x + 8

المتطابقة الثانية: مكعب فرق حدين (Cube of Difference)

(a - b)³ = a³ - 3a²b + 3ab² - b³

لاحظ الفرق بين مكعب المجموع ومكعب الفرق:

  • الإشارات تتناوب في مكعب الفرق: (+, -, +, -)
  • جميع الإشارات موجبة في مكعب المجموع: (+, +, +, +)

المتطابقة الثالثة: مجموع مكعبي حدين (Sum of Cubes)

a³ + b³ = (a + b)(a² - ab + b²)

مثال تطبيقي:

لحساب 8 + 27

  1. 8 = 2³, 27 = 3³
  2. 2³ + 3³ = (2 + 3)(2² - 2×3 + 3²)
  3. = 5(4 - 6 + 9)
  4. = 5 × 7 = 35

المتطابقة الرابعة: فرق مكعبي حدين (Difference of Cubes)

a³ - b³ = (a - b)(a² + ab + b²)

ملاحظات مهمة:

  • لاحظ الفرق بين مجموع وفرق المكعبات
  • في مجموع المكعبات: الإشارة الوسطى سالبة (-)
  • في فرق المكعبات: جميع الإشارات موجبة (+)

مثال تطبيقي 1: (Example 1)

لنحل المعادلة التالية:

4x² + 12x + 5 = 9

الحل:

  1. نجمع المتشابهات: 4x² + 12x - 4 = 0
  2. نلاحظ أن: 4x² + 12x = 4(x² + 3x)
  3. نكمل المربع: 4(x² + 3x + ╉) - 4(╉) = 0
  4. نعوض بـ ╉ = ╂²: 4(x² + 3x + ╂²) - 4(╂²) = 0
  5. النتيجة: 4(x + ╂)² - 9 = 0

مثال تطبيقي 2: (Example 2)

استخدام المتطابقة التكعيبية:

(x + 1)³

الحل:

  1. نطبق متطابقة مكعب مجموع حدين:
  2. x³ + 3x²(1) + 3x(1)² + (1)³
  3. النتيجة: x³ + 3x² + 3x + 1

نصائح للتطبيق العملي (Practical Tips)

التعرف على المتطابقة:

  • ابحث عن الأنماط المألوفة
  • حدد الحدود الجبرية الرئيسية
  • تعرف على شكل المتطابقة

تطبيق المتطابقة:

  • اكتب الحدود بشكل منظم
  • تحقق من الإشارات
  • اجمع الحدود المتشابهة

أمثلة من الحياة اليومية:

حساب مساحة الأرض:

الطول = a + b

العرض = a + b

المساحة = (a + b)²

حساب الفرق بين مربعين:

مثل: 101² - 99²

= (101 + 99)(101 - 99)

= 200 × 2 = 400

تمرين 1: (Exercise 1)

ما ناتج (x + 2)² باستخدام المتطابقة التربيعية الأولى؟

طريقة الحل:

  1. نستخدم المتطابقة: (a + b)² = a² + 2ab + b²
  2. حيث a = x و b = 2
  3. نعوض: x² + 2(x)(2) + 2²
  4. نبسط: x² + 4x + 4
أ) x² + 2x + 2
ب) x² + 4x + 4
ج) x² + 2x + 4

تمرين 2: (Exercise 2)

ما ناتج (x + 1)(x - 1) باستخدام متطابقة الفرق بين مربعين؟

أ) x² + 1
ب) x² - 2x + 1
ج) x² - 1

تمرين 3: (Exercise 3)

ما ناتج (2x - 3)² باستخدام المتطابقة التربيعية الثانية؟

أ) 4x² - 6x + 9
ب) 4x² - 12x + 9
ج) 4x² - 9
